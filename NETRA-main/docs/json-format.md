# Формат JSON

Приложение принимает объект с двумя массивами: `nodes` и `links`.

```json
{
  "nodes": [
    { "id": "srv-01", "label": "srv-01", "status": "alive" },
    { "id": "srv-02", "label": "srv-02", "status": "dead" }
  ],
  "links": [
    { "source": "srv-01", "target": "srv-02", "load": 0.73 }
  ]
}
```

## Узлы

| Поле | Тип | Описание |
|---|---|---|
| `id` | string | Уникальный идентификатор узла |
| `label` | string | Отображаемое имя узла |
| `status` | `"alive"` или `"dead"` | Состояние узла |

## Связи

| Поле | Тип | Описание |
|---|---|---|
| `source` | string | `id` исходного узла |
| `target` | string | `id` целевого узла |
| `load` | number | Нагрузка связи от `0` до `1` |

Значение `load` влияет на цвет и толщину линии. Чем ближе значение к `1`, тем выше визуальная нагрузка.
