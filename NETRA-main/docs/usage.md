# Использование

## Запуск

### Через `start.bat`

1. Откройте папку проекта `network-graph-visualizer`.
2. Дважды нажмите `start.bat`.
3. Дождитесь, пока браузер откроется автоматически.

Если порт `8000` занят, скрипт выберет следующий свободный порт в диапазоне `8000-8010` и покажет адрес в консоли. Окно `start.bat` нужно держать открытым: пока оно работает, доступно и веб-приложение.

Служебные логи запуска создаются в папке `logs/`; эта папка не добавляется в Git.

### Через `index.html`

Если Python не установлен, можно открыть `index.html` двойным кликом. Основные демо-наборы будут доступны из встроенной копии `js/demo-data.js`, а пользовательские JSON-файлы можно загрузить кнопкой «Загрузить JSON».

При таком запуске браузер может ограничивать прямую загрузку файлов из папки `data/`, поэтому для финального показа удобнее использовать `start.bat`.

## Загрузка своих данных

1. На вкладке «Главная» нажмите «Загрузить JSON».
2. Выберите файл в формате, описанном в `docs/json-format.md`.
3. После загрузки приложение переключится на вкладку «Визуализация».

После первой загрузки главная страница открывается сразу с рабочей областью. В ней доступны открытые визуализации и последние JSON, сохраненные в браузере.

## Демо-наборы

На главной странице доступны готовые сценарии:

- «Нагруженная сеть» использует `data/high-load-network.json`.
- «Много мертвых узлов» использует `data/dead-nodes-network.json`.
- «Разреженный граф» использует `data/normal-network.json`.
- «Смешанная топология» использует `data/large-network.json`.
- «Критическая сеть» использует `data/critical-network.json`.

## Работа с графом

- Перетаскивайте узлы мышью.
- Используйте колесо мыши для масштабирования.
- Кликните по узлу или связи, чтобы открыть панель деталей.
- Переключайтесь между открытыми наборами через вкладки над графом.
- Нажмите «Сбросить вид», чтобы вернуть граф в область экрана.
- Нажмите «Экспорт PNG», чтобы сохранить изображение графа.

Связи между двумя недоступными узлами отображаются красной линией. Проблемные связи с ошибками данных остаются красными пунктирными.
