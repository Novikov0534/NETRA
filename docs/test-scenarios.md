# Тестовые сценарии

Основные демонстрационные наборы лежат в папке `data/`.

| Файл | Сценарий | Назначение |
|---|---|---|
| `normal-network.json` | Нормальная сеть | Базовая проверка отображения графа |
| `high-load-network.json` | Нагруженная сеть | Проверка толщины и цвета связей при высокой нагрузке |
| `dead-nodes-network.json` | Много мертвых узлов | Проверка подсветки отказов |
| `large-network.json` | Крупная смешанная топология | Проверка читаемости графа при большем числе узлов |
| `critical-network.json` | Критическая сеть с ошибками | Проверка сообщений валидатора и подсветки проблемных связей |

Дополнительные наборы для проверки валидатора находятся в `data/validation/`.

| Файл | Что проверяет |
|---|---|
| `missing-connection.json` | Связи ссылаются на отсутствующие узлы |
| `duplicate-ids.json` | Повторяющиеся `id` узлов |
| `broken-schema.json` | Отсутствующие поля и неизвестный статус |
| `isolated-nodes.json` | Узлы без единой связи |

Проверка не блокирует отрисовку графа. Приложение показывает предупреждения, а проблемные элементы подсвечивает на визуализации.
